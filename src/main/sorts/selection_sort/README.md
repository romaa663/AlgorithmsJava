СОРТИРОВКА ВЫБОРОМ

Временная ложность алгоритма: О(n2)

Принцип: 

   1. В неотсортированном массиве ищется локальный максимум (минимум).
   2. Найденный максимум (минимум) меняется местами с первым (последним) элементом в массиве.
   3. Если в массиве остались неотсортированные элементы — продолжить сортировку, смотри пункт 1, исключив из рассмотрения уже отсортированные элементы.
